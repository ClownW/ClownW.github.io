有限自动机的三种功能：

  • 作为序列转换器，将输入序列变换为输出序列；

  • 作为序列识别器，识别输入的序列是否具有某种性质；

  • 作为序列产生器，产生具有所要求性质的序列.

例题：表示数值的字符串（DFA功能2）

由于逐步处理所有的情况非常复杂，因此考虑用有限自动机来处理。

0-8表示不同的状态，当前状态根据不同的输入转移到不同的下一个状态。

状态0：初始状态

状态1：初始状态后加上“+/-”的状态

状态2：前面是数字，可能最前面还有符号

状态3：初始状态后加上”.”后的状态

状态4：初始状态后加上小数点和数字的状态

状态5：状态2后加上小数点的状态

状态6：进入指数部分的状态

状态7：指数的数字部分

状态8：指数的符号部分

根据状态图可以列出转移矩阵：

num “+/-“ “.” e blank others
0 2 1 3 -1 -1 -1
1 2 -1 3 -1 -1 -1
2 2 -1 5 6 -1 -1
3 4 -1 -1 -1 -1 -1
4 4 -1 -1 6 -1 -1
5 5 -1 -1 6 -1 -1
6 7 8 -1 -1 -1 -1
7 7 -1 -1 -1 -1 -1
8 7 -1 -1 -1 -1 -1

根据转移矩阵进行状态转移，得到最终状态。

合法状态：{2,4,5,7}

即只有最终状态在合法状态中输入才是合法数值。
